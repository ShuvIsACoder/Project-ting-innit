# MedMystery

A Wordle-like browser game for medical students.

## What it does
- Shows 1 clue first
- You guess a disease
- If you're wrong, you unlock the next clue
- Maximum of 6 clues
- If you get it right within 6 clues, you win
- Includes 120+ diseases across multiple systems

## Files
- `index.html`
- `styles.css`
- `script.js`
- `vercel.json`

## Run locally
Just open `index.html` in your browser.

## Deploy on Vercel
### Option 1: easiest
1. Put these files in a GitHub repo
2. Go to Vercel
3. Import the repo
4. Deploy

### Option 2
Use Vercel static deployment for this folder directly.

Because this is a static site, there is no backend and no build step required.
